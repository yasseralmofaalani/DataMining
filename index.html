مادة البحث والتنقيب عن المعطيات

رحلة متكاملة تبدأ من أسس تخزين البيانات (Data Warehousing) وصولاً إلى خوارزميات التنقيب المتقدمة (Data Mining) والتطبيقات المكانية (Spatial Mining).

المحاضر: م. ياسر المفعلاني

🛠️ البرامج والتقنيات المستخدمة في الكورس

🐘 PostgreSQL (لعمليات OLTP و OLAP)
🐍 Python (Pandas, Scikit-Learn)
🐦 Weka (للتنقيب المرئي)
🌍 GeoPandas & QGIS (للتنقيب المكاني)
المحور الأول: DW

المحاضرة 1: تصميم OLTP

بناء الجداول الأساسية، العلاقات، والقيود البرمجية في PostgreSQL لضمان سلامة البيانات التشغيلية.

 المحور الأول: DW

المحاضرة 2: مستودعات البيانات

فهم Fact Tables و Dimension Tables وتطبيق Star Schema لتحويل البيانات إلى معلومات تحليلية.

 المحور الأول: DW

المحاضرة 3: عمليات ETL عملياً

استخدام Python (Pandas) لاستخراج، تنظيف، وتحميل البيانات من ملفات Excel إلى قواعد البيانات.

 المحور الأول: DW

المحاضرة 4: تحليل OLAP

تنفيذ استعلامات متقدمة باستخدام Rollup و Cube و Pivot Tables لاستخراج تقارير ديناميكية.

 المحور الثاني: Data Mining

المحاضرة 5: أساسيات التنقيب

مقدمة في المفاهيم الأساسية والفرق بين التنقيب في البيانات والتعلم الآلي والذكاء الاصطناعي.

 المحور الثاني: Data Mining

المحاضرة 6: أشجار القرار

تطبيق خوارزمية Decision Tree للتصنيف باستخدام Scikit-Learn وتقييم دقة النموذج مرئياً.

 المحور الثاني: Data Mining

المحاضرة 7: KNN & Logistic Regression

مقارنة الخوارزميات التصنيفية وتحليل Confusion Matrix لتقييم أداء النماذج البرمجية.

 المحور الثاني: Data Mining

المحاضرة 8: تجميع البيانات (K-Means)

تقسيم البيانات إلى مجموعات متجانسة آلياً وتصور النتائج باستخدام Matplotlib.

 المحور الثاني: Data Mining

المحاضرة 9: قواعد الارتباط

تحليل سلة المشتريات باستخدام خوارزمية Apriori واكتشاف الأنماط الخفية في سلوك العملاء.

 المحور الثاني: Data Mining

المحاضرة 10: كشف القيم الشاذة

استخدام Z-Score و Isolation Forest لتحديد البيانات غير الطبيعية وحالات الاحتيال.

 المحور الثالث: تطبيقات

المحاضرة 11: التنقيب المكاني

دمج أنظمة المعلومات الجغرافية (GIS) مع التنقيب باستخدام GeoPandas و DBSCAN المكاني.

 المحور الثالث: تطبيقات

المحاضرة 12: المشروع المتكامل

تطبيق عملي شامل يجمع كافة المهارات المكتسبة لبناء نظام تنبؤي وتحليلي لبيانات واقعية.

 المحور الثالث: تطبيقات

المحاضرة 13: تقنيات متقدمة

المعالجة المسبقة المتقدمة للبيانات وهندسة الميزات (Feature Engineering) لرفع كفاءة النماذج.

 المحور الثالث: تطبيقات

المحاضرة 14: العرض والنتائج

كيفية تصوير النتائج بشكل احترافي وبناء لوحات تحكم (Dashboards) تفاعلية للبيانات المنقبة.
